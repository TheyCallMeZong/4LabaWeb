## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Download project

```sh
git clone https://github.com/TheyCallMeZong/4LabaWeb
```
